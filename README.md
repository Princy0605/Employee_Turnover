# Employee Turnover Prediction

Predict employee attrition using Supervised Machine Learning techniques with Logistic Regression, Ridge Regression, and Lasso Regression.

---

## 📌 Project Overview

Employee turnover is one of the major challenges faced by organizations. High employee attrition can lead to increased recruitment costs, productivity loss, and reduced organizational efficiency.

This project builds an end-to-end **Supervised Machine Learning Pipeline** to predict whether an employee is likely to leave the company based on various employee-related features.

The project uses:

- Logistic Regression (Base Model)
- Ridge Regression (L2 Regularization)
- Lasso Regression (L1 Regularization)

The performance of all models is compared to determine the best-performing approach for employee turnover prediction.
